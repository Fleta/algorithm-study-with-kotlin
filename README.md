### 언급되지 않은 내용
 
- Kotlin을 잠깐 써 봤는데 코드가 java에 비해 많이 예쁘게 나오는 것 같음
- Kotlin이 production들에서도 많이 사용되려는 것 같아 이참에 학습하려고 함
- Kotlin에 현재 익숙하지 않다는 점을 생각했을 때 통상적으로 구현하는 것의 약 두 배의 시간이 들어갈 것이라 예상됨
